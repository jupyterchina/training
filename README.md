# Jupyter师资培训

{% hint style="info" %}
#### 教育部产学研合作项目-北京百智享&西南财经大学产学研合作项目，由百智享公司提供平台支持。
{% endhint %}

### 项目介绍

本项目基于百智享公司的教学平台，完成校内外基于Jupyter师资培训活动，本材料为结项资料总结了以Jupyter为基础的教学经验和方法；

* Jupyter基础知识；
* 计算语言支持和扩展；
* 教学模式和教学方法；
* 教学参考材料；
* 常见问题建议。
